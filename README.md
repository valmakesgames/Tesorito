# Tesoro (Equip Menu Prototype)

Tesoro means treasure in Spanish. This is a story about leaving the stars to return to Earth.

Recollect tesoros, precious memories that bring back your human form.

> "Come back to Earth, Star Child."

<br>

## Memory Wardrobe (Equip Menu)
On your journey from the stars and down back to Earth, you will unlock some of your old memories in forms of equipment items. One by one, pieces of your humanity will be restored.

![Equip Menu mockup to help define the story for the Equip Menu prototype.](https://github.com/valmakesgames/Tesorito/blob/master/art_tesorito/equip-menu_mockup_v1.png)

## Prototype Functionality
- Player can equip one item at a time from each category of equipment: Shoes, Pants, Shirts, Gloves, and Hats.
- Design can create equipment items from a menu using Scriptable Objects
- Each piece of equipment has both a weight value (0 - 999) and a defense value (0 - 9999)
- Each piece of equipment has 0 - 3 Perks
- Only one Legendary Item may be equipped at any time, attempting to equip more than one displays a warning
- When a player surpasses 70lbs total weight from equipment load, an error is shown
